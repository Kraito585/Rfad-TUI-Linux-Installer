**Установочный клиент сборки RFAD для устройств использующие OS на основе Linux.** 
___
Не является Официальным продуктом:
[**Requiem For A Dream
by Immersive Chicken**](https://skyrimrfad.com/)
___

[Не стесняйтесь открыть обсуждение если какой либо функционал не работает на вашем устройстве](https://github.com/Kraito585/Rfad-TUI-Linux-Installer/issues)

Вылет при первом запуске с ENB происходит всегда.
В ReShade будет ошибка компиляции нескольких шейдеров игнорируйте просто скройте UI ReShade.

В будующем функционал будет немного расширен.
А не стабильность google cloud будет решена переходом на зеркала CDN/S3.

Если вы видете ошибку с упоминанием JWT обязательно сообщите мне. Я решу эту проблему настолько быстро насколько это возможно.

[![image.png](https://i.postimg.cc/DfSSXLc8/image.png)](https://postimg.cc/hhq4FJ6B)

Чтобы скомпилировтаь инстайлер самостоятельно используйте архив из релиза или создайте сервисный аккаунт в google cloud включите для него "Google Drive API" сгенерируйте json api положите в src/credentials.json.
Архив из релиза уже имеет credentials.json с ограничиным доступом который можно использовать для компиляции.
