### [Не стесняйтесь открыть обсуждение если какой либо функционал не работает на вашем устройстве](https://github.com/Kraito585/Rfad-TUI-Linux-Installer/issues)

Вылет при первом запуске с ENB происходит всегда.
В ReShade будет ошибка компиляции нескольких шейдеров игнорируйте просто скройте UI ReShade.

В будующем функционал будет немного расширен.

Я решил не отказыватся от загрузки через google cloud зная какая аудитория Linux параноики.
Соответсвеноо я буду потдерживать две версии gdrive и cdn.

Кроме того откуда каждое издание скачивает файлы отличий нету все фиксы будут применятся одинакого.

---
Чтобы скомпилировтаь инстайлер создайте сервисный аккаунт в google cloud включите для него "Google Drive API" сгенерируйте json api положите в src/credentials.json.
Выполните команду:

```
go build -tags gdrive -o rfad-installer-gdrive .
```
